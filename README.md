# ⚡Spider_DCD ⚡

汽车大模型项目代码

对于非本机测试（如服务器）运行时，请检查程序Headless是否为**True**。

运行爬虫程序时请关闭所有的代理。

### ⏬Install Python Requirements

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 🚀Install Playwright

```bash
python3 -m playwright install
# or 
playwright install
```
